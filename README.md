# Superstore Sales Data Analysis Dashboard

## Project Overview
This project focuses on analyzing Superstore sales data using Excel to uncover insights about orders, profits, and business performance across the United States.

The dashboard was created to help understand:
- Sales and profit distribution across product categories
- Regional performance across the United States
- Top performing states and sub-categories
- Growth trends in sales over time
- Relationship between cost, sales, and net profit

---

## Problem Statement
The raw dataset contained thousands of sales transactions across different regions, categories, and years.

The raw dataset was difficult to analyze directly because it contained:
- Dates stored as serial numbers instead of readable format
- Floating point errors in numeric columns
- Raw data and pivot summaries mixed in the same sheet
- Inconsistent decimal precision across financial columns
- Missing values in some columns

The goal of this project was to clean and transform the dataset into meaningful visual insights that support data-driven analysis.

---

## Objectives
- Clean and preprocess raw Superstore sales data
- Analyze sales, profit, and cost trends across categories and regions
- Build an interactive dashboard using Excel
- Create dynamic visualizations for easier exploration
- Extract business insights from the dataset

---

---

## Dashboard Features
- Total Sales, Profit, and Cost KPIs
- Sales breakdown by product category
- Profit analysis by state and region
- Sub-category performance comparison
- Interactive filters and slicers

---

## Data Cleaning Process
The following preprocessing steps were applied:
- Converted date serial numbers into readable date format
- Fixed floating point precision issues in financial columns
- Separated raw data from Pivot summaries
- Added Month column for time-based analysis
- Validated calculated financial columns
- Handled missing values and formatting inconsistencies

---

## Key Insights
- Technology generated the highest sales among all product categories
- Accessories and Machines were among the most profitable sub-categories
- California achieved the highest total profit among all states
- The Central region recorded the highest sales volume overall
